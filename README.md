## SDK Version
Flutter SDK Latest

## Installing

Run this command in your terminal:
```bash
git clone
cd mboilerplateflutter
flutter pub get
flutter run
```

##### Get the packages
```bash
sh pub.sh
```

##### Get test coverage
```bash
sh gencov.sh
```

##### Get code generation test
```bash
sh build-runner.sh
```
